The main library for vidamo is in the client folder, vidamo.js.

The main librarues are:
- angular
- angular ui*
- jquery (? may not be needed)

*The angular ui library has had some small changes made to it in order to make it 
work for Vidamo.

The packages folder contains some other secondary libraries that are used by
Vidamo. they are as follows:
- ace is for the text editor
- hamster is for the interactive zooming with the mouse
- select is for the drop down menu


The structure is something like this
left pane: html <-> angular
middle pane: html <-> angular <-> threejs
right pane: html <-> angular
